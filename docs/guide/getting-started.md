# Getting Started

Welcome to the Manifesto Specification. This guide helps you run the docs locally and skim the core concepts.

## Run the Docs

1. Install dependencies: `npm install`
2. Start dev server: `npm run docs:dev`
3. Build for production: `npm run docs:build`

## Read Order

1. [Introduction](/draft/) — context and table of contents.
2. [Snapshot](/draft/2-snapshot) — the canonical state surface.
3. [Semantic Path](/draft/3-semantic-path) — addressing model.
4. [Effect](/draft/4-effect) — mutation protocol.
5. [Expression](/draft/5-expression) — declarative computation model.
6. [Validation](/draft/6-validation) and [Execution](/draft/7-execution) — runtime behavior.
